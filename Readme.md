# Portfolio Website

## 📌 About This Project
This website is developed to showcase my **portfolio**, **skills**, **projects**, and to provide more information about me in a professional and attractive manner.  
It reflects my journey, technical expertise, and commitment to continuous growth in the technology industry.

---

## 🛠️ How I Built It
This project is built using a modern development stack focusing on scalability, DevOps principles, and clean UI/UX.  
I focused not only on frontend and backend integration but also on deployment automation, ensuring the project is production-ready with CI/CD pipelines and cloud integration.

---

## 🧰 Toolstack Used
- **Node.js** - Backend environment and utilities
- **React.js** - Frontend framework for dynamic UI
- **Docker** - Containerization of the application
- **CI/CD with GitLab** - For Continuous Integration and Continuous Deployment pipelines
- **Linux** - Server configuration and shell scripting
- **Azure Virtual Machines (VM)** - Hosting the web application on cloud infrastructure
- **Email.js** - Email services integration
- **Shell Script** - Automation tasks and server-side operations
